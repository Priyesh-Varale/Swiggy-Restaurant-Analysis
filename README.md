# 🍽️ Swiggy Restaurant Analytics Dashboard

## 📌 Project Overview

An interactive Power BI dashboard designed to analyze restaurant pricing, ratings, delivery times, and city-wise restaurant performance across multiple Indian cities.

---

## 🎯 Purpose

The Swiggy Restaurant Analytics Dashboard provides valuable insights into restaurant performance, pricing trends, ratings, and delivery efficiency. It enables users to compare restaurants across different cities and identify top-performing, highly-rated, affordable, and premium restaurants through interactive visualizations.

---

## 🛠️ Tech Stack

* 📊 Power BI Desktop
* 📂 Power Query
* 🧠 DAX (Data Analysis Expressions)
* 🔗 Data Modeling
* 📈 Interactive Visualizations
* 📁 PBIX & PNG Files

---

## 📂 Data Source

**Source:** Swiggy_Final_Dataset

### Dataset Fields

* Restaurant Name
* City
* Cost for Two Persons
* Restaurant Type
* Vegetarian Availability
* Ratings
* Delivery Time
* Number of Reviews
* Restaurant Category

The dataset supports city-level analysis, rating comparisons, cost evaluation, and delivery performance monitoring.

---

## 🚀 Business Problem

Food delivery platforms host thousands of restaurants with varying prices, ratings, and delivery times.

Key questions addressed:

* Which restaurants provide the best value?
* Which cities have the highest restaurant concentration?
* Which restaurants are highly rated?
* What is the relationship between pricing and ratings?
* Which restaurants offer faster delivery services?

---

## 🎯 Dashboard Goals

* Analyze restaurant performance across major Indian cities
* Compare restaurant costs and customer ratings
* Identify premium and budget-friendly restaurants
* Evaluate delivery performance
* Support data-driven decision making

## 📈 Dashboard Features

* Mumbai
